*LibZip 1* binaries
-------------------

- Sources checked out in a branch from  [[GitHub]: nih-at/libzip - libzip](https://github.com/nih-at/libzip) (in a forked *repo* - select / checkout the appropriate tag for each version)
- Static and dynamic (shared) library variants available
- Built with *ZLib* support (also available in this *repo*). *ZLib* build type (static, dynamic) corresponds to *LibZip* build type (dynamic builds require *zlib.dll* at runtile). Builds versions:
    - ***LibZip*** -> ***ZLib***
    - *1.11.3* -> *1.3.1*

**Platforms**:
- *Windows*
